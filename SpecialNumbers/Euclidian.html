    Euclid Numbers #

    This file introduces the Euclid numbers as defined in [GKP89].

    Main results #

    • Recurrence with a product of Euclid numbers.
    • Co-primality of Euclid numbers.
    • Explicit formula.

    def euclid :
    ℕ → ℕ

    The sequence of Euclid numbers $(e_n)_{n\ge 0}$.

    Definition by a simple recurrence. The more explicit recurrence is proved in Theorem euclid_eq_prod_euclid.

    Equations

    • euclid 1 = 2
    • euclid n.succ = euclid n ^ 2 - euclid n + 1

      theorem euclid_eq_prod_euclid {n : ℕ} :
      euclid (n + 1) = ∏ x ∈ Finset.Icc 1 n, euclid x + 1

      The Euclid numbers satisfy the recurrence: $$ e_{n+1} = \prod_{i=1}^n e_i + 1. $$

      source
      theorem euclid_of_n_eq_prod_euclid {n : ℕ} (h : n ≥ 1) :
      euclid n = ∏ x ∈ Finset.Icc 1 (n - 1), euclid x + 1

      Another expression of euclid_eq_prod_euclid for easier application when $n\ge 1$: $$ e_n = \prod_{i=1}^{n-1} e_i + 1. $$

      theorem euclid_increasing {n : ℕ} :
      euclid n < euclid (n + 1)

      The Euclid numbers are strictly increasing: $e_n < e_{n+1}$, for all $n\in\mathbb{N}$.

      theorem euclid_monotone :
      Monotone euclid

      Euclid numbers form a Monotone sequence.

      theorem euclid_m_n_mod_one {m n : ℕ} (h1 : m < n) (h2 : 0 < m) :
      euclid n % euclid m = 1

      $e_n \equiv 1\ (\mathrm{mod}~e_m)$, when $0 < m < n$.

      source
      theorem euclid_rel_prime {m n : ℕ} (h : m ≠ n) :
      (euclid m).gcd (euclid n) = 1

      The Euclid numbers are co-prime: $\gcd(e_n, e_m) = 1$, for $n\neq m$.

      noncomputable def pl_euc_m (n : ℕ) :
      ℝ

      The constant in the explicit formula for the Euclid numbers comes from the limit of the sequence $\frac{1}{2^n}\log(e_n - \frac{1}{2})$.

            ↑(euclid n) ≤ euclid_constant ^ 2 ^ n + 1 / 2

            $e_n \le E^{2^n} + \frac{1}{2}$ for all $n\in\mathbb{N}$.

            theorem euclid_constant_lt_euc {n : ℕ} :
            euclid_constant ^ 2 ^ n + 1 / 2 < ↑(euclid n) + 1

            $E^{2^n} + \frac{1}{2} < e_n + 1$ for all $n\in\mathbb{N}$.

            theorem euclid_formula {n : ℕ} :
            euclid n = ⌊euclid_constant ^ 2 ^ n + 1 / 2⌋₊

            $e_n = \lfloor E^{2^n} + \frac{1}{2}\rfloor$ for all $n\in\mathbb{N}$.
